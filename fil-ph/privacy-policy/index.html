Interpretation at mga Definition

  • Mga Cookie ay mga small file na pini-place sa computer Mo, mobile device or any other device ng isang website, na nagko-contain ng mga detail ng browsing history Mo sa website na 'yon kasama ng maraming gamit nito.

  • Bansa nai-refer sa: Pilipinas

  • Device means kahit anong device na pwedeng ma-access ang Service katulad ng isang computer, isang cellphone o isang digital tablet.

  • Personal Data ay anumang information na may kaugnayan sa isang na-identify o na-identify na individual.

  • Service nai-refer sa Website.

  • Provider ng Service means ay anumang natural o legal na tao na nagpoprocess ng data para sa ngalan ng Team. Personal Data

Habang ginagamit ang Aming Service, pwede Naming i-ask na magbigay sa Amin ng certain personally identifiable information na magagamit Mo para ma-contact o ma-identify Ka. Pwedeng masama sa personal information, ngunit 'di limited sa:

  • Usage Data

Usage Data

Ang Usage Data ay na-collect automatically habang ginagamit ang Service.

Pwedeng masama sa Usage Data ang information tulad ng Internet Protocol address ng Device Mo (ex. IP address), browser type, version ng browser, mga page ng aming Service na bini-visit Mo, ang oras at petsa ng iyong pag-visit, ang oras na inii-spent sa mga pahinang 'yon, mga unique na device identifier at iba pang diagnostic data.

Pwede din Kaming mag-collect ng information na ipinadadala ng browser Mo 'pag binisita Mo ang aming Service o kapag na-access Mo ang Service sa pamamagitan ng isang mobile device.

Mga Tracking Technology at Mga Cookie

Ginagamit Namin ang mga Cookie at katulad na mga tracking technology para ma-track ang activity sa Aming Service at mag-store ng certain information. Ang mga Tracking technology na ginagamit ay mga beacon, mga tag, at mga script para ma-collect at ma-track ang information at para ma-improve at i-analyzw ang Ating Service. Ang mga Technology na ginamit Namin pwedeng may kasamang:

  • Mga Cookie o Mga Browser Cookie. Ang isang cookie ay isang small file na pini-place sa Device Mo. Pwede mong ituro ang browser Mo para ma-refuse ang lahat ng mga Cookie o para i-indicate kapag ang isang Cookie ay ipinapadala. However, 'pag 'di Mo tinatanggap ang mga Cookie, pwedeng hindi mo magagamit ang ilang mga part ng aming Service. Unless kung ina-adjust mo ang browser setting Mo para i-refuse nito ang mga Cookie, pwedeng gamitin ng aming Service ang mga Cookie.

  • Mga Flash Cookie. Ang mga certain feature ng aming Service ay pwedeng gumamit ng mga local stored objects (o mga Flash Cookie) para mag-collect at mag-store ng information about sa mga preference Mo o aktibidad Mo sa aming Service. Mga Flash Cookie ay hindi nai-manage ng parehong mga browser setting bilang mga ginagamit para sa mga Browser Cookie. Para sa more information kung paano mo matatanggal ang mga Flash Cookie, please basahin ang "Saan ko mababago ang mga setting para sa pagdi-disable, o pag-delete ng mga local shared objects?" na available sa https://helpx.adobe.com/flash-player/kb/disable-local-shared-objects-flash.html#main_Where_can_I_change_the_settings_for_disabling__or_deleting_local_shared_objects_

  • Mga Web Beacon. Ang mga certain section ng aming Service at ang aming mga email ay maaaring maglaman ng maliliit na electronic file na kilala bilang web beacon (tinukoy din bilang malinaw na mga gif, mga pixel tag, at mga single-pixel gif) na nagpe-permit sa Team, example, sa mga gumagamit na mga visitor sa mga page o na-open ang isang email at para sa iba pang related na mga website statistic (example, pagre-record ng isang certain na section at pag-verify ng system at integrity).

Ang mga Cookie ay pwedeng mga "Persistent" o "Session" Cookie. Mga Persistent Cookie ay magse-stay sa personal computer Mo o mobile device kapag ikaw ay offline, habang ang mga Session Cookie ay nai-delete sa lalong madaling panahon mo isinara ang web browser Mo. Alamin pa ang about sa mga cookie: Paggamit sa mga Cookie Free Privacy Policy.

Parehas Natin ginagamit ang mga Session at Persistent Cookie para sa mga purpose nai-set sa ibaba:

  • Necessary / Essential na mga Cookie

    Type: Mga Session Cookie

    Ina-administer ng: Tayo

    Purpose: Ang mga Cookie na ito ay essential para mag-provide ng mga service na magagamit Mo through the Website at para ma-enable Kang gamitin ang ilan sa kanyang mga feature. Tumutulong sila para ma-authenticate ang mga user at ma-prevent ang mapanlinlang na paggamit ng mga user account. Kung wala ang mga Cookie na ito, ang mga service na ina-ask mo ay hindi mapo-provide, at ginagamit lamang Namin ang mga Cookie na ito para ibigay sa Iyo ang mga serbisyong 'yon.

  • Mga Cookie Policy / Notice Acceptance na mga Cookie

    Type: Mga Persistent Cookie

    Ina-administer ng: Tayo

    Purpose: Ina-identify ng mga Cookie na'to kung tinanggap ng mga user ang paggamit ng mga cookie sa Website.

  • Mga Functionality Cookie

    Type: Mga Persistent Cookie

    Ina-administer ng: Tayo

    Purpose: Ang mga Cookie na ito ay nag-a-allow sa atin upang maalala ang mga pagpipilian na ginagawa Mo kapag ginagamit Mo ang Website, tulad ng pag-alaala sa mga detalye o kagustuhan mo sa wika. Ang purpose ng mga Cookie na ito ay para magbigay sa Iyo ng isang mas personal na karanasan at para ma-avoid ang pagkakaroon Mong ipasok ulit ng iyong mga preference kada gagamitin Mo ang Website.

Para sa more information about sa mga cookie na ginagamit namin at ang iyong mga pagpipilian about sa mga cookie, please i-visit ang aming Mga Cookie Patakaran o ang Mga Cookie section ng aming Privacy Policy.
